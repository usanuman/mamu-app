# Cronicle

Cronicle is a multi-server task scheduler and runner with a web-based UI. It handles both time-based (cron) and on-demand jobs across multiple servers, with real-time stats and a live log viewer.

## Features

- Schedule jobs using standard cron syntax
- Run jobs across multiple servers in your homelab
- Real-time job output and live log viewer
- Built-in retries, notifications, and job chaining
- REST API for triggering jobs programmatically
